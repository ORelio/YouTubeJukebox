===============
 Mode d'emploi
===============

Il vous suffit d'ouvrir l'application et de saisir le nom d'utilisateur de la chaîne
Ensuite, cliquez sur le bouton Play pour lancer la récupération de la liste des vidéos ainsi que la lecture.

==================================
 Utilisation en ligne de commande
==================================

En particulier sur Mac et Linux, vous pouvez utiliser YouTube Jukebox avec le framework Mono :

$ mono YouTubeJukebox.exe <Chaîne YouTube>

Utilisez mono YouTubeJukebox.exe --help pour obtenir la liste des arguments acceptés.
