============
 How to use
============

Just launch the application and type in the username of the channel
Then, press the Play button to fetch the video list and launch it in the media player.

====================
 Command-line usage
====================

Especially on Mac and Linux, you can use YouTube Jukebox with Mono framework:

$ mono YouTubeJukebox.exe <YouTube channel>

See mono YouTubeJukebox.exe --help for a list of allowed arguments
